```txt
───────┬─────────────────────────────────────────────────────────────────────────────────
       │ File: config.yaml
───────┼─────────────────────────────────────────────────────────────────────────────────
   1   │ config:
   2   │   install_tree:
   3   │     root: /home/artdaq/DAQ_SPACK_DevAreas/DAQ_2..._artdaqv3_13_02/spack_packages
   4   │     padded_length: 255
───────┴─────────────────────────────────────────────────────────────────────────────────
```
